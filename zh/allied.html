盟军

阵营

由美国和西欧国家主导建立的联盟，拥有强大的空军和高精度武器。盟军在光学领域拥有强大的技术积累，落地了隐形发生器、光学棱镜聚能炮等多项技术。

盟军建筑

盟军建造场

建造场是所有前线建设的基础。建造场不可以直接被建造，而是由机动基地车直接展开部署而得。

  • 收起部署为盟军机动基地车。

建造前提：无

能耗： 0

费用：2000（基地车）

盟军发电厂

发电厂是战场中基础的电力供应设备，没有电力，前线的武器生产和防御设施将会停摆。盟军发电厂是传统火力发电厂的代表，其发电机组内部充满氢气，因此在被摧毁时会引发剧烈的氢气爆炸。

  • 被摧毁后引发氢气爆炸。
  • 被渗透后停电。

建造前提：盟军建造场

能耗：+150

费用：800

盟军矿石精炼厂

矿石精炼厂可以将战场上宝贵的矿石精炼为资源，用于生产建筑和武器，以及换取经济资源。

  • 建造完成时提供一辆隐形采矿车，采矿车无法停放时返还资金。
  • 被渗透后敌方获得资金。

建造前提：建造场 + 发电厂

能耗：-50

费用：2000

盟军兵营

兵营是前线的基础训练设施，可以训练步兵。

  • 被渗透后敌方步兵获得初始升级。

建造前提：建造场 + 发电厂

能耗：-10

费用：1000

盟军武器工厂

武器工厂是前线的基础生产设施，可以制造装甲车辆以及直升机。

  • 被渗透后敌方载具获得初始升级。

建造前提：盟军建造场 + 矿石精炼厂 + 兵营

能耗：-50

费用：2000

盟军作战控制中心

作战控制中心是前线的必要指挥设施，其搭载了一整套通讯设备和雷达装置，可以提供雷达并且呼叫来自本土的侦察机。

  • 提供侦察机战术。
  • 提供雷达。
  • 被渗透后重置战争迷雾。

建造前提：建造场 + 矿石精炼厂

能耗：-50

费用：1000

盟军空指部

盟军空指部可以生产以及维护战斗机，同时为战斗机的行动提供导航。

  • 提供空降步兵战术。
  • 被渗透后敌方飞行器获得初始升级。

建造前提：建造场 + 矿石精炼厂 + 作战控制中心

能耗：-50

费用：1000

盟军作战实验室

盟军作战实验室可以为战场前线提供更高的科技，解锁新的建筑、防御和武器。

  • 被渗透后敌方获得隐藏单位。

建造前提：建造场 + 矿石精炼厂 + 作战控制中心

能耗：-100

费用：3000

矿石提纯器

矿石提纯器可以在采集到的矿石中提纯出更多高价值的原料，让每一车矿石换取更多的资源。

  • 采矿带来的收入提升25%。

建造前提：建造场 + 矿石精炼厂 + 作战实验室

能耗：-100

费用：2500

盟军防御

围墙

围墙可以有效阻挡敌军的步兵和轻型载具突袭。

建造前提：盟军建造场

能耗：0

主武器：无

费用：50

机枪碉堡

机枪碉堡是盟军的基础防御工事，可以有效消灭冲锋的敌军士兵和轻型载具。

建造前提：

  1. 盟军建造场
  2. 盟军兵营

能耗：0

主武器：M2重机枪

弹种：穿甲弹

费用：500

防空导弹

盟军在掌握了热跟踪导弹技术后，很快将其落地到了实战。一次性可以发射8枚热跟踪导弹，防空导弹对任何空军的威胁都是致命的。

建造前提：

  1. 盟军建造场
  2. 盟军兵营

能耗：-50

主武器：FIM-43

弹种：高爆弹

费用：1000

光棱塔

光棱塔是盟军的终极防御武器，其拥有一个非常强大的特性——光子增幅。每一座光棱塔可以利用棱镜组将光束聚焦到一座塔上，高能光束的汇集可以形成一股超高能光束，对来犯的敌人进行毁灭性打击。

  • 多个光棱塔串联后提升攻击力。

建造前提：建造场 + 兵营 + 作战控制中心

能耗： -50

主武器：光学棱镜组

弹种：光子射流

费用：1500

隐形装置

隐形装置是盟军的高级防御工事，当通电时，隐形装置可以在周围生成一道光学屏障，让一切附近的单位获得隐形。但是，隐形装置无法让自身隐形。

  • 可以为范围内所有单位提供隐形。
  • 自身无法隐形。

建造前提：建造场 + 兵营 + 作战实验室

能耗：-100

费用：2000

卫星通讯器

卫星通讯器可以利用卫星连接与本土的空军指挥部进行通讯，并为战术轰炸机提供导航，提高命中精度。

  • 提供堡垒轰炸战术。
  • 提供雷达。
  • 被渗透后重置战术倒计时。

建造前提：建造场 + 作战控制中心 + 作战实验室

能耗：-100

费用：2000

天气控制仪

天气控制仪是盟军的终极大规模杀伤性武器。天气控制仪通过聚焦光束发射到对流层，改变对流层分子运动趋势，并在指定坐标形成闪电风暴。

  • 提供闪电风暴战术。
  • 可以通过客户端的【超级武器】选项关闭。
  • 被渗透后重置战术倒计时。

建造前提：建造场 + 作战控制中心 + 作战实验室

能耗：-200

费用：4000

盟军战术

空降步兵

空降步兵是基础的战术之一，通过空投一个步兵班到战场的任意地方，在不经意间袭扰敌军。

  • 提供4名突击兵, 4名重装兵。

使用前提：空指部

费用：500

加载时间：120秒

堡垒轰炸

SBX-47对流层堡垒轰炸机是盟军的超音速低空轰炸机，速度极快，普通的防空武器很难对付。对流层堡垒轰炸机每次轰炸任务会携带5枚高爆炸弹，投向目标区域。记住，在轰炸任务开始后一定要让友军撤离目标区域，炸弹的杀伤力不容小觑。

  • 每次投放5枚CBU-32高爆炸弹。

使用前提：卫星通讯器

费用：2000

冷却时间：120秒

闪电风暴

闪电风暴形成后，将对地面的一切物体造成毁灭性打击。

  • 闪电风暴可以让雷达失效。

使用前提：天气控制仪

费用：0

冷却时间：120秒

盟军步兵

突击兵

突击兵是盟军的基础步兵。突击兵装备了M16自动步枪，并且在部署后可以使用M1919重机枪以提供更强大的火力。同时，突击兵具备巷战能力，可以进驻建筑以作为掩体。

  • 可以进驻建筑。
  • 可以驾驶中立载具。
  • 部署后使用重型机枪。

训练前提：兵营

主武器：M16自动步枪

费用：150

重装兵

重装兵是盟军的重火力支援步兵。重装兵装备了M60轻机枪，并且在部署后可以使用反坦克火箭筒以打击敌方载具。和突击兵一样，重装兵也可以进驻建筑以作为掩体。

  • 可以进驻建筑。
  • 部署后使用反坦克火箭筒，并无法碾压。

训练前提：兵营

主武器：M60轻机枪

费用：300

工程师

工程师是战场上最重要的角色之一，其不仅可以修复建筑，还可以占领敌方建筑和中立建筑。工程师最强大的武器不是枪而是知识，因此在战场上需要被重点保护。

  • 可以维修己方建筑。
  • 可以维修桥梁。
  • 可以占领敌方和中立建筑。
  • 可以拆除C4炸弹。

训练前提：兵营

主武器：工具箱

费用：500

狙击兵

狙击兵可以精准命中步兵的弱点进行远程射击。同时，狙击兵可以将轻型载具中的驾驶员射杀后使其停滞并由己方步兵驾驶。更甚，狙击兵可以通过窗户对建筑中进驻的步兵进行清除。

  • 可以射杀轻型载具驾驶员。
  • 可以射杀进驻建筑的步兵。

训练前提：兵营 + 作战控制中心

主武器：M40狙击步枪

费用：1000

特种兵

特种兵主要由海军的海豹突击队组成，擅长两栖作战和渗透破坏任务。特种兵装备了MP5微型冲锋枪，同时携带C4炸药，可以轻松炸毁敌军建筑。

  • 可以渡水。

训练前提：兵营 + 作战实验室

主武器：MP5微型冲锋枪

费用：1000

间谍

间谍是在战场阴暗面活跃的谍报人员，其擅长渗透，并可以窃取敌军科技、毁坏敌军电力、干扰敌军雷达，或者篡改敌军战术。

  • 可以渗透敌方建筑。

训练前提：兵营 + 作战控制中心

主武器：伪装道具

费用：1000

盟军载具

盟军机动基地车

M57机动基地车是由皮特比尔特重载车头改装而来，其拥有强大的越野能力。同时，盟军机动基地车还加装了装甲和防弹玻璃，使其很难被轻武器所击毁。

  • 部署后展开为盟军建造场

建造前提：武器工厂 + 战场控制中心

主武器：无

弹种：无

费用：2000

隐形采矿车

HV41隐形采矿车是盟军所开发的一种新型矿车，其搭载了盟军的第二代隐形装置，可以在低能耗的情况下实现隐形功能。同时，因为采用了轻型坦克的底盘，隐形采矿车拥有强劲的机动性，但载矿量则有些差强人意。

  • 可以隐形。
  • 被摧毁后留下矿箱。

建造前提：矿石精炼厂 + 武器工厂

主武器：无

弹种：无

费用：1400

灰熊坦克

M54A2灰熊坦克是盟军的新型中型坦克，其装备了一门90毫米M5线膛炮，并拥有厚重的装甲。M54A2灰熊坦克是由M54坦克发展而来，但摒弃了不成熟的105毫米主炮，而采用了相对稳定的90毫米主炮。对比前代M26潘兴坦克，M54灰熊坦克的武器、机动性和装甲都更胜一筹。

  • 可以在移动中开火。

建造前提：盟军武器工厂

主武器：90毫米M5线膛炮

弹种：穿甲弹

费用：1600

多功能步兵车

M21多功能步兵车是盟军最好的多功能装甲车。其主要装备为一个6管防空导弹发射器，这让多功能步兵车成为了敌军空中单位的噩梦。同时，多功能步兵车首次尝试了模块化设计，可以轻松切换炮塔，因此不同的乘员可以切换不同的炮塔在战场上发挥不同的作用。

  • 跟随乘员切换炮塔和武器。

建造前提：武器工厂

主武器：MIM-21地空导弹

弹种：高爆弹

费用：1200

隐形坦克

M51隐形坦克是盟军的战术型单位。因为搭载了隐形发生器，所以隐形坦克的全部空间都用来放置一台大功率发电机，这也导致了隐形坦克不得不采用一门4连装105毫米M45无后坐力炮。隐形坦克可以用一轮齐射消灭绝大多数单位，但因为装填时间过于漫长，你必须谨慎选择目标。

  • 可以隐形，开火后解除隐形。

建造前提：武器工厂 + 作战控制中心

主武器：105毫米M45无后坐力炮

弹种：破甲弹

费用：2000

晓鹰直升机

UH-0晓鹰直升机是盟军最成功的运输直升机之一，其拥有超大的内部空间，可以容纳5名士兵的同时，士兵可以在机舱内开火。同时，晓鹰直升机的机头处有一部小型雷达，可以真查到地面上的隐形单位。

  • 乘员可以对外开火。
  • 可以探测隐形。

建造前提：武器工厂 + 空指部

主武器：无

弹种：无

费用：2000

光棱坦克

T67PMC光棱坦克是盟军的新型秘密武器。光棱坦克搭载了一个小型化的光子射流器，可以通过光子棱镜炮将光束引向指定目标，在集中目标后光束还会进行散射，点燃范围内的其他敌人。

  • 击中目标后形成光子散射。

建造前提：武器工厂 + 作战实验室

主武器：光子棱镜炮

弹种：光子射流

费用：2400

战斗要塞

T111战斗要塞是盟军一直在秘密研究的终极武器。战斗要塞装备了一门恐怖的155毫米T11线膛炮，其强大的火力可以重创任何敌人，甚至可以将大部分敌人一发秒杀。同时，战斗要塞搭载了相当厚重的装甲，这也让战斗要塞成为盟军最理想的突击载具。

  • 可以碾压载具。
  • 可以在外挂装甲被击毁后爆甲。

建造前提：武器工厂 + 盟军作战实验室

主武器：155mm T11

弹种：穿甲弹

费用：3000

短剑战机

YF-102D短剑战机是盟军中速度最快的空中力量。装备有两枚AIM-4C空空导弹，短剑战机可以轻松拿到制空权。同时，短剑战机还装备有一枚空地导弹，可以应急对付敌军装甲部队的袭击。

  • 可以进行空战。

建造前提：空指部 + 作战控制中心

主武器：AIM-4C空空导弹

弹种：高爆弹

费用：2000

入侵者攻击机

XA-6入侵者攻击机是盟军最新的亚音速攻击机，其可以装载两枚Mk-77凝固汽油滑翔炸弹，可以摧毁几乎任何地面单位，同时在目标区域制造燃烧，高温的地表持续灼烧路过的敌人。

  • 可以在目标地表制造高温区域。

建造前提：空指部 + 作战实验室

主武器：Mk-77凝固汽油弹

弹种：凝固汽油弹

费用：2500
