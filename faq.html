Technical Questions

Does Bellum Æternum contains Yuri's Revenge?

No. Bellum Æternum is an unofficial modification (mod) of Command & Conquer Red Alert 2 Yuri's Revenge, and since the original game isn't free, you need to get it first before installing Bellum Æternum.

Why the client won't run?

Running the client needs .NET platform, so if your PC is with Windows 10 or earlier version of Windows, you can go to the Microsoft .NET download center，click the download button under Run desktop apps, and install it。

If your PC is with Windows 8.x or earlier version, you need to additionally install the Microsoft Visual C++ 2015-2019，64 bit or 32bit depending on your processor.

Why the game crashes while launching skirmish or campaign?

Bellum Æternum should be installed upon Red Alert 2 Yuri's Revenge version 1.001, so please make sure the original game is in v1.001.

You can run the game by ra2md.exe and check the version number in the bottom-left corner of the main screen.

General Questions

Is Bellum Æternum free?

Bellum Æternum is absolutely free to play, and there is NO in-game content you need to pay for.

How can I set Bellum Æternum client to English?

There is possibility the client isn't English as default, you can go to the settings panel by clicking the settings (with a gear icon) button on the main screen, then click the Language (中文) to swap it.
