# Welcome to Our Documentation!

RA2DIY Tutorial Document.

## Features:

* **Static Site:** No complex server needed.
* **Markdown Powered:** Write your content in easy-to-read Markdown.
* **HTMX Reusability:** Header, footer, and navigation are loaded dynamically.
* **Client-Side Rendering:** Markdown is converted to HTML in the browser.

### Getting Started

To view this site, simply open `index.html` in your web browser. For local development, it's best to use a simple local web server (e.g., Python's `http.server` or Node's `serve`) to avoid CORS issues when fetching `.md` files.
