Bellum Æternum Installation Guide

Navigation

When you have finished all content about Red Alert 2, you might find there are dozens of high-quality mods with brand new content. However, after you have downloaded some of them, you might stuck and wondering how to install those mods for Red Alert 2 or Yuri's Revenge.

Since a lot of players are completely new to Red Alert 2 / Yuri's Revenge mods, this guide is about how to install mods for Red Alert 2 and Yuri's Revenge.

Facts about Red Alert 2 mods

Before you jump to the installation procedure, there are several facts you need to know:

  1. Most mods are developed based on Yuri's Revenge, not the vanilla Red Alert 2, because the game engine of Yuri's Revenge has more features. Therefore, you need to make sure what game does the mod requires before installation.
  2. Most modern mods, which developed after 2015, have integrated Ares, so you shouldn't launch the game by YURI.exe or RA2MD.exe. Instead, you can run the game by double-clicking RunAres.bat file.
  3. Nowadays, more mods have integrated their own client based on CNCNet client. Therefore, you should carefully read the docs of the mod.
